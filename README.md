# BlueFarm Monitoring System

Vue 3 기반으로 재구성하는 B2B 내부관리 웹서비스 포트폴리오 프로젝트입니다.  
양식장 환경 데이터를 모니터링하고 장치 및 수조 상태를 관리하는  
IoT 대시보드형 관리자 시스템 UI를 구현하는 것을 목표로 합니다.

## Overview

- B2B 내부관리 웹서비스 UI를 Vue 3 기반으로 재구성
- IoT 모니터링 대시보드 구조 설계
- 지도 기반 양식장 상태 모니터링 UI
- 수조 및 장치 상태 관리 인터페이스 구현
- 센서 데이터 기반 실시간 차트 UI 구성

## Tech Stack

- HTML5 / CSS3
- JavaScript (ES6+)
- Vue 3 (Composition API)
- Vite
- Vue Router
- Pinia
- Chart Library (ECharts / Chart.js 예정)
